# libraryMVIP
本人MVIP模式 + FragmentationX 组成

MVIP模式: 由MVP模式演变而来, M + IV + IP 以工厂模式实现P统一化, V映射

FragmentationX:
一个功能强大的库，用于管理Android的Fragment！
"单Activity ＋ 多Fragment","多模块Activity + 多Fragment"架构而生，简化开发，轻松解决动画、嵌套、事务相关等问题。
